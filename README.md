# Diseño de pipeline para rutas de senderismo:captura multi fuente transformación e interfaz dual.
Este proyecto implementa técnicas de data engineering para un sistema completo de gestión de rutas de senderismo, resolviendo la fragmentación de información mediante un pipeline automatizado que integra múltiples fuentes. La arquitectura modular abarca nueve componentes con mecanismos de idempotencia y monitorización exhaustiva. 
El diferencial principal es el doble enfoque de consumo: API REST para integración programática y dashboard Streamlit con interfaz dual que ofrece herramientas analíticas para expertos y recomendaciones personalizadas para usuarios finales. Los resultados validan un sistema funcional de extremo a extremo, procesando miles de rutas con trazabilidad completa y generando métricas avanzadas de dificultad e intensidad. La solución demuestra la viabilidad de construir plataformas escalables de data engineering con tecnologías open-source, cumpliendo requisitos de captura multi-fuente, transformación y disponibilización diferenciada.
